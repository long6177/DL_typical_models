<h2 align="center">深度学习一些经典模型</h2>

关于深度学习领域, 主要NLP和CV两个方面的一些经典论文模型

每个文件夹为一个模型, md文件为学习笔记, **涵盖模型的基本知识**
当前文件夹的[问题思考.md](./问题思考.md), 是对一些经典模型的一些典型问题的理解与思考
